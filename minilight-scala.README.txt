Description
-----------

MiniLight is a minimal global illumination renderer. See the main MiniLight
readme for a general description.




Building
--------

Requirements:
* Scala 2.9.1 or later installed
* scala/bin directory is on path

It is an ordinary simple build. There are no special requirements or
dependencies: everything needed is in the supplied archive (assuming the build
environment and tools are already prepared).

For Mac or Linux, run the make/build.sh script. For Windows, run the
make/build.bat script.

Run scripts from the 'scala' directory.




Usage
-----

Requirements:
* Scala 2.9.1 or later installed
* scala/bin directory is on path

Run, from the 'scala' directory, like so:
   scala -classpath obj hxa7241.minilight.MiniLight
